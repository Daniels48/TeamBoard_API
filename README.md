# TeamBoard

Kanban board application built with FastAPI.

## Features

- JWT Authentication
- Redis Sessions
- User Roles (Admin/User)
- Boards
- Columns
- Cards
- Drag & Drop
- Board Layout Saving
- PostgreSQL
- Alembic Migrations
- Docker Compose

## Tech Stack

- Python 3.12
- FastAPI
- PostgreSQL
- SQLAlchemy
- Redis
- Alembic
- Docker

## Run

```bash
docker compose up --build
